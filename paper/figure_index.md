**Supplementary Figure 4. Triage-agent reason agreement with the curator is high overall and concentrated in within-bucket reassignments, with the Sonnet 4.6 + NCBI production configuration scoring 87.1 % exact reason agreement on the 147-gene SurfaceBench.** Three-panel composite that walks the agent's `predicted_reason` against the curator's `ground_truth_reason` (both drawn from the same closed 19-value `TriageReason` enum) at three resolutions. **(a) Bucket-strict accuracy across all 10 model-variant cells** тАФ for every (Haiku 4.5, Sonnet 4.6, Opus 4.8) ├Ч (naive, + IDs, + IDs + web, + IDs + PubMed) cell available on mainbench (Opus runs only naive + IDs), bar height is the mean-of-replicates fraction whose predicted-reason bucket matches the curator reason's bucket (Yes / Contextual / No). The grouping is strict тАФ yes тЖФ contextual mismatches are *not* soft-credited тАФ so this is a tougher accuracy axis than Supp Fig 1. Overall bar (with dotted separator) plus per-bucket bars; overlaid dots are per-replicate accuracies and error bars are SEM. **(b) Per-reason accuracy across 4 frontier configs** тАФ best-Haiku (web + IDs), Sonnet 4.6 (+ IDs and + IDs + PubMed), best-Opus (+ IDs). Each bar group is one reason from the 16-of-19 codes that appear in the bench, ordered left-to-right Yes тЖТ Contextual тЖТ No with x-tick labels colored by bucket (green / orange / grey) for at-a-glance bucket-stratification. **(c) Confusion matrix for the production configuration** тАФ Sonnet 4.6 + NCBI predictions collapsed across three replicates by mainbench_canonical_v2's majority-then-first-rep rule, with rows / columns blocked by surface-verdict bucket (separated by light grey dividers); the diagonal carries 128 / 147 genes (87.1 % exact agreement) with red cell outlines, and off-diagonal cells with тЙд 3 genes additionally label the genes by name (the largest single bucket is cytoplasmic тЖТ inner_leaflet_anchored at 3 / 19: BTK, JAK1, AKT2 тАФ a within-`no` reassignment that does not affect verdict accuracy).

**Figure caption.** Topology composition of SurfaceBench (n = 97 of 147 that join into the per-protein features TSV) compared against the Sonnet 2-tier yes/contextual universe (n = 4,426), the genes the production pipeline ships as accessible after the NCBI sweep + PubMed rescue lane. Bars show the % of each subset that carries the topological feature; error bars on SurfaceBench are Wilson 95% binomial confidence intervals (the universe is a population, not a sample, so no CI). Per-class p-values are computed by an exact 2-tailed binomial test of the bench count against the universe proportion under HтВА, Bonferroni-corrected across the 9 topology classes (* p < 0.05, ** p < 0.01, *** p < 0.001); 3 of 9 classes reach significance. The bench is enriched for Glyc. site (+20.1 pp*), GPI- anchored (+14.5 pp*), Single-pass TM (+11.6 pp) and depleted for Inner-leaflet lipidated (-0.3 pp), No TM / no signal (-0.6 pp), Multi-pass TM (-11.0 pp*); this reflects its deliberate selection for DB-disagreement cases, so bench-derived accuracy estimates are a lower bound on expected full-universe accuracy.
