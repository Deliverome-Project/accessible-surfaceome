# scripts/figures/

Canonical generators for the figures published in the paper companion. Each
script here is paired with a `data/analysis/figures/make_<slug>.py` mirror
that ships in a public gist for reader-side reproduction. Per CLAUDE.md's
canonical-vs-mirror rule, both files must be kept in sync — edit each side
in the same commit and regenerate the rendered `<slug>.pdf` + `<slug>.png`
in `data/analysis/figures/` so the on-disk outputs match.
