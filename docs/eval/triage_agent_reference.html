Task context (per-gene) B2M sample

In addition to the system prompt, every run gets a per-gene task message populated by the orchestrator: HGNC approved name + symbol, UniProt accession, aliases / previous symbols, and the NCBI gene summary. We don't inject UniProt's curated subcellular_locations — that would over-bias toward UniProt's dominant-compartment call.

Alternative prompts opt-in

Three prompt variants live in prompts/. The active default is v2 (system.md). The others are checked in as artifacts for opt-in experiments.

Why three? The original system.md was strictly weaker than v2 on the KAAG1 / TGFB1 edge cases under Haiku and Sonnet. The web-search variant system_websearch.md closes the remaining Haiku gap (Haiku on TGFB1 = GARP-tethering, a training-data gap) but costs ~5–7× more per protein. Use v2 for the default genome-wide pass; reserve web-search for borderline second-pass.
