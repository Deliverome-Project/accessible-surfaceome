**Figure 2. A tool-free Accessibility Triage agent classifies hard surface proteins more accurately than any single database.** Classification accuracy on SurfaceBench, 147 proteins enriched for disagreement among the five source databases and manually labeled as surface, contextually surface, or non-surface. The leftmost group reports accuracy across all proteins; the rest break it out by label. Bars compare the triage agent (Sonnet 4.6, given canonical gene identifiers but no external tools or web search) against the five databases. Contextually surface proteins count as correct when called either surface or contextual. Error bars indicate SEM across replicates; deterministic database calls are shown without error bars.
