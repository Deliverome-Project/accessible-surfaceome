Task context (per-gene) ERBB2 sample

Each run gets a per-gene task message with two halves: a one-line annotate-the-gene instruction, then the ## Pre-loaded deep-dive context block injected by the orchestrator from deep_dive_pack.render_markdown. The injected block carries DeepTMHMM topology for human isoforms and mouse/cyno orthologs, plus Ensembl Compara identity — so the agent never needs to call a tool just to populate the deep-dive fields.

Pydantic schema

The agent emits a SurfaceomeRecordDraft JSON; the orchestrator promotes each EvidenceClaim to a full Evidence record (substring-anchored against cached source bodies) and persists a canonical SurfaceomeRecord. The deep-dive additions (IsoformAccessibility, CoreceptorRequirement, OrthologRecord) are additive — empty lists are valid for genes without isoform / co-receptor / ortholog data.
