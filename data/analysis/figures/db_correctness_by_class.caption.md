**Figure 2. A tool-free Accessibility Triage agent classifies hard surface proteins more accurately than any single database.** Classification accuracy on SurfaceBench, 147 proteins enriched for disagreement among the five source databases and manually labeled as surface, contextually surface, or non-surface (n = 68, 31, and 48). The leftmost group reports accuracy across all 147 proteins; the remaining groups break it out by label. Bars compare the triage agent (Sonnet 4.6, given canonical gene identifiers but no external tools or web search) against the five databases. Contextually surface proteins are counted correct when called either surface or contextual. The agent was run in 3 replicates; error bars indicate SEM across replicates. Database calls are deterministic and shown without error bars. [Figure data and script.](https://gist.github.com/beccajcarlson/2bb4f7aac629535982c142bc2032e04d)
