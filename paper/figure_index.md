## Figure 4 — `deep_dive_flow`

- **Gist:** [gist](https://gist.github.com/beccajcarlson/eb7ce1015ff21ec0329472e33a9ea713) · `swh:1:rev:506d29ab3052576e6db9b33782e55b5d975483de`
- **Bundled TSV:** _(SVG mockup — no data TSV)_

**Figure 4. The per-gene deep-dive pipeline runs in three sequential stages, producing one validated record.** Triage-candidate genes flow top to bottom; box color encodes the engine performing each step. Stage 1, evidence retrieval (deterministic, no LLM): public databases build a candidate corpus, a per-paper Haiku 4.5 abstract triage selects papers worth fetching, tiered open-access full-text recovery is attempted, and a Sonnet 4.6 selector yields a trimmed evidence ledger. Stage 2, block builders (Sonnet 4.6): a concurrent fan-out emits surface-evidence, surface-biology, evidence-grade, and cross-focus risks blocks. Stage 3, synthesizer reads every block draft plus the Stage 1 ledgers, deterministic features, and triage prior, then emits the executive summary, structured filters, and confidence rationale. Each validated record is published at roughly $2–3 per gene.

- **Gist:** [gist](https://gist.github.com/beccajcarlson/f724dd328f48566a354f0294109a7337) · `swh:1:rev:1af3cb5977fcc5da28a55dc8dde57240622eb103`

**SurfaceBench over-samples the membrane topologies where databases disagree, so its measured accuracy is a conservative floor on full-catalog performance.** The per-class composition of the hand-curated benchmark is compared against the full shipped accessible set across nine membrane-topology classes, with each class tested by an exact two-tailed binomial test of the benchmark count against the population proportion, Bonferroni-corrected across the nine classes. - **Gist:** [gist](https://gist.github.com/beccajcarlson/152a4d8b4a1d156e2afaa73d205294b6) · `swh:1:rev:63180540e9cdd254a9491da4dbcfce5b0b046b0a`
- **Bundled TSV:** `triage_vs_deep_dive_reason.tsv` (20 rows · 1.1 KB)

**On the small set of genes carried through both stages, the deep-dive's evidence-anchored reason matches the triage agent's first-pass reason about half the time, with the remaining disagreements split between same-verdict relabelling and outright verdict changes.** Each gene gets a short reason for why it was or was not called a surface protein, drawn from one shared vocabulary used at both stages; triage assigns it from abstracts alone, while the deep-dive re-derives it after reading full text. Reasons are grouped into surface-yes, context-dependent, and surface-no verdict blocks, so a reason swap that keeps the verdict is distinguishable from a changed verdict. Only twenty genes currently have records from both stages, so this is an early, low-count comparison rather than a stable rate.
